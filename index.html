Sobre a Calculadora

Projeto de construção de uma calculadora simples e leve para ser usada em páginas HTML, feita com elementos HTML e JavaScrip, a calculadora simples html tem as seguintes caracteristicas:

  • Possui duas telas (display).
  • Um apresenta todas as operações calculadas, com uso de barra de rolagem..
  • A a segunda tela apresenta o resultado processado.
  • A precedência dos dos operadores é feita da esquerda para a direita com o uso de parênteses automáticos.
  • Possui botão extra "R" que serve para calcular o resto da divisão.
  • As demais caracteristicas são semelhantes a calculadoras tradicionais.

O designer da calculadora foi elaborado de forma bem simples usando um formulário, uma tabela e elementos HTML comuns.

 

Funções Importantes a Serem Implantadas na Calculadora

  • Porcentagem
  • Raiz Quadrada
  • Potência
  • Botão +/-
  • Botão "C" (apaga somente ultimo numero)
  • Memória "M" e "MR"(memoriza e recupera um numero)
